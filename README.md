# Flow  
Flow is an extremely simple app which has one job, and that's to remind you to drink water!

## Features
- System notifications with random motivating lines to drink water
- Ability to set custom intervals between each notification
- Extremely clean and simple UI
- Works in the background when minimized to not take taskbar space
## Download
1. [Download Flow.zip](https://github.com/Evil-Rat/Flow-Water-Reminder/releases/download/v1.0/Flow.zip)
2. Extract the zip folder wherever you like
3. Open Flow.exe
4. (Optional : create a desktop shortcut for quick access)
5. Done!
